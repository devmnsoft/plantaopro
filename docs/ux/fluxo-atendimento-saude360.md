# Fluxo de Atendimento Saúde 360

A página `ClinicaDashboard/FluxoAtendimento` guia a operação em nove passos: paciente, agendamento, check-in, painel, triagem, consulta, prescrição, financeiro e relatórios.

Cada card possui explicação simples, status e botões reais para telas do sistema, sem `href="#"`.

## Teste rápido
Acesse o Web, faça login como administrador e abra `ClinicaDashboard/FluxoAtendimento`. Siga os cards em ordem para validar a jornada leiga.
