# Roteiro de homologação — Administrador Global

- Acessar dashboard global e validar clientes ativos, clientes em teste, receita prevista, faturas vencidas, chamados críticos e erros do sistema.
- Validar que filtros globais não misturam ações entre clientes sem indicação clara.
- Acessar Central de Escala e confirmar visão consolidada multiempresa.
- Testar acesso a clientes, planos, assinaturas e faturamento SaaS.
- Confirmar auditoria para ações críticas e tentativas de acesso negado.
