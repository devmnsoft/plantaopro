# UI Guidelines Mobile

## Cores
- Primária: azul institucional do PlantãoPro.
- Sucesso: verde.
- Atenção: âmbar.
- Erro: vermelho.

## Tipografia
- Escala simples: 12 / 14 / 16 / 20 / 24.
- Priorizar contraste AA.

## Componentes
- Cards com status e CTA primária.
- Botões grandes (altura mínima 44px).
- Ícones semânticos por contexto (agenda, convite, pagamento).

## Telas principais
- Login
- Dashboard
- Plantões disponíveis
- Convites
- Escalas
- Pagamentos
- Notificações
- Perfil e Preferências

## Experiência mobile-first
- Uma ação principal por tela.
- Empty states com orientação objetiva.
- Feedback visual de carregamento e sucesso.
