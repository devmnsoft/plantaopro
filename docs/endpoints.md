# Endpoints
A API expõe autenticação, dashboard, CRUD de médicos/hospitais/especialidades/plantões, escalas, financeiro, notificações e relatórios.

## Endpoints desta rodada
- GET/POST/PUT/DELETE /api/hospitais
- GET/POST /api/especialidades
- GET/POST /api/plantoes e ações /publicar /cancelar /encerrar
- POST /api/plantoes/{id}/aceitar
- POST /api/financeiro/pagamentos/gerar
- GET /api/notificacoes
- GET /api/dashboard e /api/mobile/home
