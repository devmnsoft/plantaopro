# v1.13 — Runtime E2E

O job `runtime-e2e-v113` sobe PostgreSQL, aplica migration e seed, executa API em background e roda `scripts/smoke-test-v113.sh` contra endpoints reais.

O smoke aceita HTTP 401 como evidência de autenticação protegida quando não há token de homologação configurado. Quando `PLANTAOPRO_TOKEN` está presente, o token é usado no header e nunca é impresso.
