# Checklist LGPD e auditoria

Escopo: PlantãoPro Saúde 360.

- [ ] Senha, token, hash e API key nunca são logados.
- [ ] Dados clínicos sensíveis não entram em log técnico.
- [ ] Acesso a resumo clínico é auditado.
- [ ] Impressão de prescrição é auditada.
- [ ] Exportação é auditada.
- [ ] Recepção não acessa evolução clínica.
- [ ] Financeiro não acessa anamnese.
- [ ] Médico acessa apenas seu contexto permitido.
- [ ] Admin cliente respeita tenant.
- [ ] Admin global é auditado.
