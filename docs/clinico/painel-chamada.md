# Painel de chamada

O painel opera filas por tenant, unidade, setor, sala e guichê. O modo TV usa token público seguro na tabela `paineis_chamada` e deve respeitar white label do tenant.

## Status

AGUARDANDO, CHAMADO, RECHAMADO, EM_ATENDIMENTO, FINALIZADO, AUSENTE e CANCELADO.

## Segurança

Configurações permitem exibir somente senha para reduzir exposição de dados pessoais na sala de espera.
