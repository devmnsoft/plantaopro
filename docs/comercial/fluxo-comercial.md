# Fluxo Comercial

Lead (NOVO/CONTATADO/QUALIFICADO) -> Oportunidade (ABERTA/NEGOCIAÇÃO/PROPOSTA) -> Proposta (RASCUNHO/ENVIADA/APROVADA) -> Contrato (ATIVO) -> Onboarding -> Implantação.

## Regras
- Sem duplicidade por CNPJ/e-mail quando possível.
- Perda/recusa exigem motivo.
- Aprovação de proposta notifica admin global.
- Eventos críticos devem ser auditados.
