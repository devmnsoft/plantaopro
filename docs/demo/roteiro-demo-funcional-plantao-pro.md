# Roteiro demo funcional PlantãoPro

1. Login admin demo.
2. Abrir Fluxo de Atendimento.
3. Cadastrar paciente sem digitar GUID.
4. Criar agendamento por lookup de paciente e médico.
5. Fazer check-in, chamar no painel, executar triagem e abrir consulta.
6. Buscar CID, criar prescrição, gerar conta a receber e receber pagamento.
7. Cadastrar convênio/plano e demonstrar plantões/escalas.
