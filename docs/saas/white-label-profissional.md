# White label profissional

White label é acessível para admin global e admin do tenant. O layout usa logo, tenant, plano, slogan, favicon e CSS de tema. A tela de configuração deve mostrar status, plano atual, recursos disponíveis/bloqueados, preview, última alteração e ações de editar, aplicar template e publicar.

## Fase 1 — aplicação visual

O layout usa variáveis `--tenant-primary` e `--tenant-secondary` com fallback seguro para o tema PlantãoPro. A sidebar, logo, plano e tenant já deixam o white label visível; persistência avançada de assets permanece pendente.
