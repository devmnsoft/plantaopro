# Bloqueios por plano e módulo

Bloqueios usam `_ModuleLockedNotice`, `_PlanLimitReachedNotice` e `_UpgradeCta`, com motivo claro, plano atual, plano necessário, uso atual, limite contratado, link para assinatura, suporte e comercial. A API retorna motivo funcional em vez de exception técnica para o usuário final.
