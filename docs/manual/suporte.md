# Manual do Suporte

## O que o perfil faz
Atende chamados, acompanha incidentes, consulta observabilidade permitida e orienta administradores do cliente sem acessar conteúdo clínico sensível.

## Menus disponíveis
- Ajuda e Governança
- Suporte
- Observabilidade permitida pelo contrato

## Jornada principal
1. Receber chamado.
2. Confirmar tenant, perfil impactado e módulo.
3. Registrar evidências sem senha, token, prescrição, anamnese ou diagnóstico.
4. Encaminhar para produto/engenharia quando necessário.
5. Comunicar solução e registrar encerramento.

## Limites de permissão
Não alterar dados clínicos, não acessar evolução clínica, não solicitar senha e não exportar dados pessoais sem base legal.

## Dúvidas frequentes
- Quando escalar incidente? Quando houver indisponibilidade, erro 500 recorrente, risco LGPD ou impacto financeiro.
- Como pedir logs? Solicitar janela de data/hora e usuário, mantendo mascaramento de dados sensíveis.

## Como pedir suporte interno
Registrar severidade, tenant, rota, horário UTC, usuário afetado, evidência e impacto operacional.
