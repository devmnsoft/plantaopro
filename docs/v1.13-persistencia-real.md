# v1.13 — Persistência real operacional

A v1.12 era um fluxo demonstrativo com estado em memória. A v1.13 transforma o módulo em fluxo homologável com PostgreSQL, Dapper, autenticação, tenant e respostas `ApiResponse<T>`.

## Itens entregues

- Migration idempotente para tabelas `plantaopro.v113_*`.
- Seed demo seguro sem senha, token, JWT ou connection string sensível.
- `V113OperationalService` centralizando acesso Dapper/Npgsql.
- Controller de compatibilidade com rotas antigas e rotas `/api/v113/*`.
- Outbox real em `v113_outbox_eventos` e `v113_outbox_logs`.
- Boleto demonstrativo em `/api/v113/billing/titles/{id}/demo-boleto`, com mensagem sem valor financeiro real.

## Pendências reais

- Validar performance com volume de dados maior.
- Ligar worker real de entrega externa somente após homologação e parametrização segura.
- Ampliar telas para formulários ricos por módulo.
