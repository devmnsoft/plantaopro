# Jornada do cliente leigo

## Princípio
Cada tela deve responder: onde estou, para que serve, o que fazer agora e qual o próximo passo.

## Jornada atendimento
1. Cadastrar paciente.
2. Criar agendamento.
3. Realizar check-in.
4. Chamar no painel.
5. Iniciar triagem.
6. Iniciar consulta.
7. Vincular CID.
8. Criar prescrição quando aplicável.
9. Gerar conta a receber.
10. Receber pagamento.

## Diretrizes
- Botão principal único por tela.
- Empty state com explicação e próxima ação.
- Ajuda contextual sempre visível.
- Linguagem sem jargão técnico.
