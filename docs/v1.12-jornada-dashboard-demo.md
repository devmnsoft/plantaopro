# v1.12-jornada-dashboard-demo

Entrega v1.12 adiciona endpoints funcionais de homologação para clientes, produtos, estoque, pedidos, tarefas, faturamento, outbox, templates, dashboard, jornada, atividades, demo e homologação visual.

## Como validar

1. Executar build/test em ambiente com SDK .NET.
2. Subir API e Web apontando `PlantaoProApi:BaseUrl` para a API.
3. Executar `scripts/smoke-test-v112.ps1`.
4. Abrir `/api/homologation/status`, `/api/demo/status`, `/api/dashboard` e `/api/journey/what-to-do-now`.

## Limitações conhecidas

O container local não possui `dotnet`; portanto a evidência local de build/test está documentada em `docs/v1.12-build-test-report.md`.
