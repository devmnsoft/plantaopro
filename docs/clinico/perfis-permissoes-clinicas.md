# Perfis e permissões clínicas

Perfis consolidados: RECEPCAO, TRIAGEM, ENFERMAGEM, COORDENADOR_CLINICO, ADMINISTRADOR_CLINICA, AUDITOR_CLINICO e MEDICO.

## Módulos

- PACIENTES: Visualizar, Criar, Editar, Inativar, VerResumoClinico, Exportar.
- AGENDAMENTO: Visualizar, Criar, Editar, Cancelar, Reagendar, Confirmar, CheckIn.
- PAINEL_CHAMADA: Visualizar, Configurar, Chamar, Rechamar, Ausente, Finalizar.
- TRIAGEM: Visualizar, Criar, Editar, Iniciar, Finalizar, Cancelar, VerHistorico.

A migration popula permissões base em `clinica_permissoes_modulos` com plano mínimo por permissão.
